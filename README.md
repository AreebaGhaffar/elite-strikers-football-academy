## 🛠️ How to Edit Website Content (Admin)

> 🔑 **First change the password:** 
- Open `admin.html` in any text editor
- find the line `var ADMIN_PASSWORD = "elite2024"` 
- change `elite2024` to your desired password. 
- Save the file.

1. Open `admin.html` in your browser
2. Enter the password to login
3. Edit any section — hero, about, programs, coaches, footer
4. Click **Save and download content.json**
5. Place the downloaded `content.json` file back into the project folder (same folder as `index.html`)
6. Refresh the website — changes appear instantly
